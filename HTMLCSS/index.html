CSS

CSS es un lenguaje de hojas de estilo (no de programacióno), sirviendo para darle propiedades a los elementos de HTML para asignarle un diseño. Un ejemplo claro es el diseño actual de este proyecto, que se encuentra todo diseñado con css.

A continuacón una tabla con los elementos escenciales de css.


Tabla con propiedades/atributos de CSS

Atributo Descripción
Margin Establece un margen al rededor de un elemento (de forma externa).
width Establece el ancho de un elemento
