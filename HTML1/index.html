Biblia de etiquetas

para agregar algun espaciado, puedes usar la eetiqueta "br" para forzarr un espaciado

si deseas agregar textos en negritas, puedes usar la etiqueta "b", el cual pondra en negritas todo aquel texto dentro de esa etiqueta

Igualmente pedes usar la etiqueta "strong" para colocar en negritas un texto, su principal función es dar más enfasis sobre algo, lo que ayuda al navegador/buscador web a buscar aquel enfasis.

La etiqueta "i" funciona para colocar letra "italica".

Tambien podemos agregar cistas con la etiqueta "cita" que se usa mayormente para agregar autores como: Rodrigo David Cañas Pérez

Links.

Existe una etiqueta que permite colocar enlaces la cual es "a" teniendo el siguiente formato:

a href=""

acontinuación un ejemplo de la etiqueta:

Ir a Youtube

dentro del "href" se coloca el link al que quieres que te redirija: href="https://www.youtube.com/"

La etiqueta, por default, carga el link en la misma página, pero si queremos que abra otra página para no perder la original, puedes hacerlo de la siguiente forma: target="_blank" href="https://www.youtube.com"
